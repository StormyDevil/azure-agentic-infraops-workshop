## Before the Event

Complete everything in [pre-work-checklist.md](pre-work-checklist.md):

- [ ] Docker Desktop installed
- [ ] VS Code with Dev Containers extension
- [ ] Git configured
- [ ] Azure subscription ready
- [ ] GitHub Copilot access verified

## During the Event

1. Read [scenario-brief.md](scenario-brief.md) for context
2. Follow the challenges in order
3. Use [hints-and-tips.md](hints-and-tips.md) if stuck (try first!)

Good luck! 🚀
